# Практична робота №2

## Тема
JavaScript. Динамічний контент

## Варіант
Варіант 11: Компенсаторна установка (КРМ)

## Опис
Проєкт реалізує веб-сторінку моніторингу енергетичного об'єкта. Дані параметрів генеруються JavaScript, оновлюються вручну кнопкою або автоматично таймером, а стан параметрів позначається кольорами: зелений - норма, жовтий - попередження, червоний - критично.

## Структура
```text
index.html
css/styles.css
js/script.js
README.md
```

## Функціонал
- Динамічне оновлення параметрів КРМ
- Ручне оновлення кнопкою
- Автоматичне оновлення кожні 3 секунди
- Візуальна індикація статусів
- Історія вимірювань у localStorage
- Графік коефіцієнта потужності cos φ
- Експорт історії в CSV
- Темна/світла тема
- Адаптивний дизайн

## Запуск
Відкрити `index.html` у браузері або запустити через Live Server у VS Code.
