# Практична робота №3: Реєстр ЕСКО контрактів

Варіант 11: **ЕСКО контракти**

Проєкт реалізує HTML-форму для збору даних про енергосервісні договори та серверну обробку за допомогою Node.js + Express. Дані зберігаються у JSON-файлі `data/esco_contracts.json`.

## Функціональність

- форма додавання ЕСКО контракту;
- базова клієнтська та серверна валідація;
- збереження даних у JSON;
- перегляд списку зареєстрованих контрактів;
- пошук і фільтрація за статусом;
- розрахунок строку окупності та тривалості договору;
- статистика по кількості, активних договорах і річній економії;
- видалення записів.

## Запуск

```bash
npm install
npm start
```

Після запуску відкрити у браузері:

```text
http://localhost:3000
```
